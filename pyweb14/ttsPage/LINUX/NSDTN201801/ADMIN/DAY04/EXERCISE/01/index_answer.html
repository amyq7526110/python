1 简述yum客户端配置文件中主要参数的含义

参考答案

[root@server0 ~]# cat  /etc/yum.repos.d/rhel_dvd.repo 
[rhel_dvd]  							//仓库标识
gpgcheck = 0 							//不做签名检查
enabled = 1 							//启用此仓库
baseurl = http://content.example.com/rhel7.0/x86_64/dvd
  									//软件仓库的访问地址

2 如何为RHEL7服务器设置固定的主机名

参考答案

使用hostnamectl命令设置：

[root@server0 ~]# hostnamectl  set-hostname  新主机名

直接修改/etc/hostname文件，重启后生效：

[root@server0 ~]# vim  /etc/hostname
新主机名
[root@server0 ~]# reboot

3 简述为网络连接“System eth0”设置IPv4地址的命令行操作

参考答案

[root@server0 ~]# nmcli  con  mod  "System eth0"  ipv4.method manual  ipv4.addresses  "IPv4地址/掩码长度  默认网关" 
[root@server0 ~]# nmcli  connection  up  "System eth0"
Connection successfully activated (D-Bus active path: /org/freedesktop/NetworkManager/ActiveConnection/1)

4 使用find命令将目录/boot下大于10M的文件，复制到/opt目录

参考答案

 [root@server0 ~]# find  /boot  -size  +10M  -type f  -exec  cp  {}  /opt   \;
